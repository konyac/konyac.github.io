第一篇

发表于 2016-05-28   |   分类于 Diary   |  

尝试每天写点东西，随手搭了一个hexo的博客。感觉还不错，有待继续挖掘。基础的功能应该够用了。
先记录一些常用的命令：

hexo new “postName” #新建文章
hexo new draft “new_draft” #新建草稿
hexo publish [layout] #可以把草稿变成文章，或者页面
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo s -p 5000 #在本地的5000 端口预览
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本
hexo clean #清除
hexo g&&hexo d #合并发布
